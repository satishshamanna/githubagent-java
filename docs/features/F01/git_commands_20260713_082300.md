# Git Deployment Commands Reference - Feature F01

Reference commands list to guide staging, committing, and pushing the US01 implementation files to the feature branch.

---

## Staging & Committing Code Changes

```bash
# Verify modified and untracked files
git status

# Stage database updates
git add src/main/resources/db/h2/schema.sql
git add src/main/resources/db/h2/data.sql

# Stage model entity updates
git add src/main/java/org/springframework/samples/petclinic/model/Appointment.java

# Stage repository implementations
git add src/main/java/org/springframework/samples/petclinic/repository/AppointmentRepository.java
git add src/main/java/org/springframework/samples/petclinic/repository/jpa/JpaAppointmentRepositoryImpl.java
git add src/main/java/org/springframework/samples/petclinic/repository/springdatajpa/SpringDataAppointmentRepository.java
git add src/main/java/org/springframework/samples/petclinic/repository/jdbc/JdbcAppointmentRepositoryImpl.java

# Stage service layer updates
git add src/main/java/org/springframework/samples/petclinic/service/ClinicService.java
git add src/main/java/org/springframework/samples/petclinic/service/ClinicServiceImpl.java

# Stage controller & binder configurations
git add src/main/java/org/springframework/samples/petclinic/web/VetFormatter.java
git add src/main/java/org/springframework/samples/petclinic/web/AppointmentController.java
git add src/main/resources/spring/mvc-core-config.xml

# Stage user interface files
git add src/main/webapp/WEB-INF/jsp/owners/ownerDetails.jsp
git add src/main/webapp/WEB-INF/jsp/appointments/createOrUpdateAppointmentForm.jsp

# Stage automated tests
git add src/test/java/org/springframework/samples/petclinic/web/AppointmentControllerTests.java
git add src/test/java/org/springframework/samples/petclinic/service/AbstractClinicServiceTests.java

# Stage documentation and agent specifications
git add .agents/AGENTS.md
git add docs/features/F01/
git add docs/functional/index.md

# Commit all changes with a descriptive message referencing US01
git commit -m "feat(US01): implement vet appointment scheduling owner flow"
```

---

## Pushing to Remote Repository

```bash
# Push to the feature branch
git push origin feature/F01-appointment-scheduling
```
