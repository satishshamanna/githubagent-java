# Feature F01 Release Notes - Version 1.0.0

Release notes covering the implementation details and packaged components for Feature F01, User Story 01.

---

## 1. Feature Overview: Appointment Scheduling & Vet Availability (US01)
This release delivers the Owner Flow functional capability allowing pet owners to schedule veterinarian appointments for their pets. Key user capability enhancements include:
- A new dedicated appointment booking button under Owner details next to each registered pet.
- A dynamic scheduling form utilizing Flatpickr datepicker selecting dates starting tomorrow.
- Veterinarian filtering based on veterinarian specialties (e.g. radiology, surgery, dentistry).
- Real-time double-booking checking restricting selections to non-conflicting time slots.
