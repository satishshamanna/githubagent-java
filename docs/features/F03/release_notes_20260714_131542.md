# Feature F03 Release Notes - Version 1.0.0

Release notes covering the implementation details and packaged components for Feature F03, User Story 01.

---

## 1. Feature Overview: Clinic Billing & Invoicing (US01)
This release delivers the Staff Flow functional capability allowing receptionist and clinic administrators to manage pet billing. Key user capability enhancements include:
- A new dedicated "Billing" tab in the navigation menu bar linking to `/admin/invoices`.
- A receptionist billing dashboard listing all invoices in the system sorted by issue date (newest first).
- Visual status badges to track UNPAID and PAID invoices.
- An invoice creation form with owner dropdown selection, dynamic flatpickr datepickers, and field validation checks (e.g. positive amount constraints, valid due dates).
