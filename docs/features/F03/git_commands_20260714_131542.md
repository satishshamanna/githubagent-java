# Git Deployment Commands Reference - Feature F03

Reference commands list to guide staging, committing, and pushing the US01 implementation files and regression testing configurations to the feature branch.

---

## Staging & Committing Code Changes

```bash
# Verify modified and untracked files
git status

# Stage database updates
git add src/main/resources/db/h2/schema.sql
git add src/main/resources/db/h2/data.sql

# Stage model entity updates
git add src/main/java/org/springframework/samples/petclinic/model/Invoice.java

# Stage repository implementations
git add src/main/java/org/springframework/samples/petclinic/repository/InvoiceRepository.java
git add src/main/java/org/springframework/samples/petclinic/repository/jpa/JpaInvoiceRepositoryImpl.java
git add src/main/java/org/springframework/samples/petclinic/repository/springdatajpa/SpringDataInvoiceRepository.java
git add src/main/java/org/springframework/samples/petclinic/repository/jdbc/JdbcInvoiceRepositoryImpl.java

# Stage service layer updates
git add src/main/java/org/springframework/samples/petclinic/service/ClinicService.java
git add src/main/java/org/springframework/samples/petclinic/service/ClinicServiceImpl.java

# Stage controller configurations
git add src/main/java/org/springframework/samples/petclinic/web/InvoiceController.java

# Stage user interface files
git add src/main/webapp/WEB-INF/tags/menu.tag
git add src/main/webapp/WEB-INF/jsp/invoices/invoiceList.jsp
git add src/main/webapp/WEB-INF/jsp/invoices/invoiceForm.jsp

# Stage automated tests
git add src/test/java/org/springframework/samples/petclinic/web/InvoiceControllerTests.java
git add src/test/java/org/springframework/samples/petclinic/service/AbstractClinicServiceTests.java

# Stage global pipeline and agent rule updates
git add instructions.md
git add project_specs.md
git add .github/agents/05-test.agent.md

# Stage documentation and agent specifications
git add docs/features/F03/

# Commit all changes with a descriptive message referencing US01 and regression configuration
git commit -m "feat(US01): implement billing flows and configure regression testing in test agent"
```

---

## Pushing to Remote Repository

```bash
# Push to the feature branch
git push origin feature/F03-clinic-billing
```
