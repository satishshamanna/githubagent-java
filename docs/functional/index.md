# Functional Reference Index

This registry tracks the active business and functional capabilities of the Spring PetClinic application.

## Active Capabilities

### 1. Owners and Pets Management
- **Description**: Allows adding, updating, and viewing owners and their pets.
- **Traceability**: Default core feature.

### 2. Veterinarian Services
- **Description**: Displays a list of active veterinarians and their corresponding specialties.
- **Traceability**: Default core feature.

### 3. Visit History Logging
- **Description**: Logs past medical visits, dates, and vet notes for pets.
- **Traceability**: Default core feature.

### 4. Appointment Scheduling & Vet Availability (F01)
- **Description**: Enables active booking of future time slots with specific veterinarians filtered by specialty.
